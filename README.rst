==================
Switchboard Python
==================

[Switchboard](http://thusfresh.github.io/switchboard) Python provides
helpers for writing Switchboard workers and clients in Python.


Examples
========

All examples are located under `/examples`.


Installation
============

This should get you running::

    # Building
    ./setup.py build
    # Running the tests
    ./setup.py test
    # Development install
    pip install -e .
    # Actual install
    pip install .
    # To run the tests
